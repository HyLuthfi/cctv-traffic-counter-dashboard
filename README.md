# TrafficVision

Sistem Penghitung Kendaraan Berbasis Computer Vision dengan Dashboard Interaktif.

## Struktur Proyek
- `backend/`: API dan Computer Vision Engine (FastAPI, YOLOv8)
- `frontend/`: Web Dashboard (React, Vite, Leaflet)

## Kebutuhan Sistem
- Python 3.11+
- Node.js 18+
- Docker & Docker Compose
